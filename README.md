# Ola_Ride_Data_Analysis_Project
Ola Ride Data Analysis Tools Used: Excel, SQL, Power BI

- Analyzed Ola ride booking data to uncover insights on ride success, cancellations, payment methods, and customer behavior
- Created complex SQL queries to extract KPIs such as top customers, average ratings, and booking value
- Built an interactive Power BI dashboard to visualize trends and operational metrics
- Identified a 28% cancellation rate and highlighted improvement areas using data-driven insights
